# Conversation Knowledge Mining Solution Accelerator

## Source

- Repository: https://github.com/microsoft/Conversation-Knowledge-Mining-Solution-Accelerator

## What it is

An accelerator for deriving insights from large conversational datasets using Microsoft Foundry, Azure Content Understanding, and Azure OpenAI.

## Best fit

- Contact center analytics
- Conversation trend analysis
- Key phrase and topic extraction from conversations

## Core capabilities

- Conversational insight extraction
- Topic modeling and key phrase analysis
- Interactive chat and exploration experience

## Adoption notes

Use when your target is conversation intelligence and operational insight, not only assistant Q&A.
