# Content Generation Solution Accelerator

## Source

- Repository: https://github.com/microsoft/content-generation-solution-accelerator

## What it is

An accelerator for marketing and content generation workflows using a multi-agent system on Microsoft Foundry.

## Best fit

- Marketing content pipelines
- Brand-aligned content generation
- Multi-step creative workflows

## Core capabilities

- Multi-agent content generation flow
- Foundry-powered orchestration
- Structured generation process

## Adoption notes

Choose this when your top priority is content productivity and consistent output workflows.
