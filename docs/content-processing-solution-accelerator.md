# Content Processing Solution Accelerator

## Source

- Repository: https://github.com/microsoft/content-processing-solution-accelerator

## What it is

A programmatic content processing accelerator for extracting data and applying schemas to text and multimodal documents.

## Best fit

- Document ingestion pipelines
- Structured extraction from unstructured content
- Multimodal content processing

## Core capabilities

- Content extraction and schema mapping
- Azure AI Foundry + OpenAI integration
- Cosmos DB-based processed output patterns

## Adoption notes

Use this when your first problem is transforming source content into structured, retrievable data.
