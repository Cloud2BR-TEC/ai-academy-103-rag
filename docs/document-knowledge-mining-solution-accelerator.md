# Document Knowledge Mining Solution Accelerator

## Source

- Repository: https://github.com/microsoft/Document-Knowledge-Mining-Solution-Accelerator

## What it is

A solution accelerator that processes and extracts summaries, entities, and metadata from unstructured multimodal documents, enabling search and chat over that content.

## Best fit

- Document intelligence programs
- Enterprise search and chat over document corpora
- Knowledge extraction from complex files

## Core capabilities

- Summary/entity/metadata extraction
- Azure OpenAI + Azure AI Document Intelligence integration
- Searchable knowledge mining output

## Adoption notes

Use for document-heavy enterprise use cases where extraction quality is critical before RAG response generation.
