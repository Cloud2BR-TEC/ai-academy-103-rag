# Modernize Your Code Solution Accelerator

## Source

- Repository: https://github.com/microsoft/Modernize-your-code-solution-accelerator

## What it is

An accelerator designed to help teams modernize and translate SQL queries across dialects and environments.

## Best fit

- Data estate modernization
- SQL migration projects
- Legacy query translation

## Core capabilities

- SQL translation acceleration
- Environment transition support
- Modernization workflow guidance

## Adoption notes

Use when a major dependency for AI adoption is modernizing legacy data/query layers.
