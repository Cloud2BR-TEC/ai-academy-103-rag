## What it solves
Creates a unified intelligence layer connecting enterprise data, knowledge, and decision support across Work IQ (Microsoft 365), Foundry IQ (AI/agents), and Fabric IQ (analytics).

## Tech stack
| Component | Role |
|-----------|------|
| Microsoft 365 | Workplace data source |
| Azure AI Foundry | AI agent and reasoning layer |
| Microsoft Fabric | Analytics and data layer |
| Azure OpenAI | Language model for synthesis |

## Adoption notes
Use when your target goes beyond a single chat assistant and you need intelligence spanning productivity, AI, and analytics in a unified experience.
