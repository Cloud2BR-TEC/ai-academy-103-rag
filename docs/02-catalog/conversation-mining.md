## What it solves
Derives insights from large volumes of conversational data — call logs, chat transcripts, support tickets — providing key phrase extraction, topic modeling, sentiment analysis, and interactive chat over the resulting insights.

## Tech stack
| Component | Role |
|-----------|------|
| Microsoft Foundry | Agent orchestration |
| Azure AI Content Understanding | Semantic analysis |
| Azure OpenAI | Insight synthesis |
| Foundry IQ | Insight layer |

## Adoption notes
Pair with Customer Chatbot accelerator to close the loop: chatbot generates conversations, this accelerator mines them for quality signals.
