## What it solves
A comprehensive RAG solution accelerator with common enterprise requirements and best practices for chat over private organizational data — from document ingestion to grounded chat with source attribution.

## Tech stack
| Component | Role |
|-----------|------|
| Azure AI Search | Document retrieval |
| Azure OpenAI | Grounded response generation |
| Azure Functions | Orchestration and ingestion |
| Azure App Service | Chat frontend |
| Microsoft Entra ID | User authentication |

## Adoption notes
The broadest-scope RAG starter template from Microsoft. Use it as the foundation and extend for your specific data sources, security requirements, and UX needs.
