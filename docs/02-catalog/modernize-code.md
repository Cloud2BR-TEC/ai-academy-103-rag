## What it solves
Helps teams transition SQL queries from legacy dialects (Oracle, Teradata) to modern Azure environments at scale, using AI to analyze syntax and produce translated, validated queries.

## Tech stack
| Component | Role |
|-----------|------|
| Azure OpenAI | SQL analysis and translation |
| Azure SQL Database | Primary target environment |
| Microsoft Fabric | Analytics target environment |

## Adoption notes
SQL translation quality should always be validated by a data engineer before production deployment.
