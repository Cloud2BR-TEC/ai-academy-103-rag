# Customer Chatbot Solution Accelerator

## Source

- Repository: https://github.com/microsoft/customer-chatbot-solution-accelerator

## What it is

A solution accelerator for building intelligent customer service chat experiences using Microsoft Foundry agent capabilities.

## Best fit

- Customer support modernization
- Conversational self-service channels
- Assisted support with contextual responses

## Core capabilities

- Agent-based conversation orchestration
- Customer interaction workflows
- Enterprise integration readiness

## Adoption notes

Use this accelerator when customer experience and support operations are your primary outcome.
