# Multi-Agent Custom Automation Engine

## Source

- Repository: https://github.com/microsoft/Multi-Agent-Custom-Automation-Engine-Solution-Accelerator

## What it is

An AI-driven multi-agent reference solution that coordinates multiple specialized agents to complete complex tasks based on user input.

## Best fit

- Workflow automation with multiple agent roles
- Task decomposition and coordinated execution
- Process orchestration scenarios

## Core capabilities

- Multi-agent orchestration
- Agent Framework + Azure Foundry integration
- Cosmos DB-backed state handling

## Adoption notes

Use when one agent is not enough and you need role-based, coordinated agent behavior.
