# Chat with Your Data Solution Accelerator

## Source

- Repository: https://github.com/Azure-Samples/chat-with-your-data-solution-accelerator

## What it is

A RAG solution accelerator that includes common enterprise requirements and best practices for chat and Q&A over private data.

## Best fit

- Internal assistant experiences
- Rapid enterprise RAG proof-to-production path
- Teams adopting a practical reference baseline

## Core capabilities

- End-to-end RAG implementation pattern
- Azure AI Search retrieval + Azure OpenAI generation
- Common enterprise controls and best-practice defaults

## Adoption notes

This is a strong starting point when teams want a practical, supported RAG template to adapt quickly.
