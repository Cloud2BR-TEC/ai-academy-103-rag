# Azure GPT-RAG

## Source

- Repository: https://github.com/Azure/GPT-RAG

## What it is

GPT-RAG is a secure enterprise Retrieval-Augmented Generation pattern on Azure using Azure AI Search for retrieval and Azure OpenAI for Chat/Q&A experiences.

## Best fit

- Enterprise knowledge assistant
- Internal policy and document Q&A
- Controlled and secure AI search/chat deployments

## Core capabilities

- Azure AI Search retrieval pipeline
- Azure OpenAI response generation
- Secure architecture patterns for enterprise deployment

## Adoption notes

Start here if your priority is a production-ready RAG baseline with strong security controls and known Azure architecture patterns.
