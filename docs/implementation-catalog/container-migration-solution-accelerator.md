# Container Migration Solution Accelerator

## Source

- Repository: https://github.com/microsoft/Container-Migration-Solution-Accelerator

## What it is

A multi-agent migration assistant for moving container service configurations from other cloud platforms to Azure Kubernetes Service.

## Best fit

- Container platform modernization
- Cross-cloud migration analysis
- AKS target architecture migration

## Core capabilities

- AI-assisted migration guidance
- Multi-agent migration workflow
- AKS-focused migration acceleration

## Adoption notes

Use when migration complexity is high and teams need guided AI-assisted transition to AKS.
