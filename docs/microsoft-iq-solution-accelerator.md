# Microsoft IQ Solution Accelerator

## Source

- Repository: https://github.com/microsoft/microsoft-iq-solution-accelerator

## What it is

A unified intelligence layer that connects enterprise data, knowledge, and decision support across Work IQ, Foundry IQ, and Fabric IQ concepts.

## Best fit

- Unified enterprise intelligence initiatives
- Cross-domain decision support
- Knowledge and data unification

## Core capabilities

- Single intelligence layer pattern
- Multi-source data and knowledge integration
- Enterprise decision support architecture

## Adoption notes

Use when your target is broader than chatbot UX and includes enterprise decision intelligence.
